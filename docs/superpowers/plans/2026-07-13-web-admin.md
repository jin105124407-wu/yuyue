> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first usable desktop web admin for the WeChat appointment mini program.

**Architecture:** Add a dedicated `webAdminApi` cloud function for phone/password web admin authentication and admin data operations. Add a static `web-admin/` browser app that calls the API, renders orders/services/staff, and polls voice notices for speaker playback.

**Tech Stack:** Native CloudBase cloud function with `wx-server-sdk`, vanilla HTML/CSS/JavaScript, existing CloudBase collections.

- [ ] **Step 2: Implement cloud function**

Implement:

```js
exports.main = async (event = {}) => {
  const input = parseEvent(event);
  if (input.action === 'login') return login(input);
  const admin = await verifyToken(input.token);
  if (!admin) return { ok: false, reason: '登录已过期，请重新登录' };
  return dispatchAdminAction(input, admin);
};
```
